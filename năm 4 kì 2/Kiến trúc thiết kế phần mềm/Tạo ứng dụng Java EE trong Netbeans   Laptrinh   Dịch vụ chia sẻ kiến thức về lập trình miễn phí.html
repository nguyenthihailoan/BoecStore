    Tạo ứng dụng Java EE trong Netbeans

    Trong tut này, mình sẽ hướng dẫn các bạn tạo và chạy một ứng dụng Java EE trong Netbeans, cũng như sửa một số lỗi hay gặp.

    Chuẩn bị

    • IDE: Netbeans 8.0.2
    • Web Server: GlassFish 4.1
    • Hệ quản trị CSDL: MySQL
    • Hệ điều hành: Windows 7

    Sửa lỗi thường gặp khi dùng GlassFish 4.1

    Với phiên bản GlassFish 4.1, ta rất dễ gặp lỗi như này khi deploy EJB:

    Ta sẽ sửa lỗi này theo các bước như sau:

    glassfish-resource.xml

    Các bạn mở file glassfish-resources.xml trong Server Resources của module Demo-ejb:

    <pre><code class="xml"><?xml version="1.0" encoding="UTF-8"?>
    <!DOCTYPE resources PUBLIC "-//GlassFish.org//DTD GlassFish Application Server 3.1 Resource Definitions//EN" "http://glassfish.org/dtds/glassfish-resources_1_5.dtd">
    <resources>
    <jdbc-connection-pool allow-non-component-callers="false" associate-with-thread="false" connection-creation-retry-attempts="0" connection-creation-retry-interval-in-seconds="10" connection-leak-reclaim="false" connection-leak-timeout-in-seconds="0" connection-validation-method="auto-commit" datasource-classname="com.mysql.jdbc.jdbc2.optional.MysqlDataSource" fail-all-connections="false" idle-timeout-in-seconds="300" is-connection-validation-required="false" is-isolation-level-guaranteed="true" lazy-connection-association="false" lazy-connection-enlistment="false" match-connections="false" max-connection-usage-count="0" max-pool-size="32" max-wait-time-in-millis="60000" name="mysql_bookstore_rootPool" non-transactional-connections="false" pool-resize-quantity="2" res-type="javax.sql.DataSource" statement-timeout-in-seconds="-1" steady-pool-size="8" validate-atmost-once-period-in-seconds="0" wrap-jdbc-objects="false">
    <property name="serverName" value="localhost"/>
    <property name="portNumber" value="3306"/>
    <property name="databaseName" value="bookstore"/>
    <property name="User" value="root"/>
    <property name="Password" value="root"/>
    <property name="URL" value="jdbc:mysql://localhost:3306/bookstore?useUnicode=true&characterEncoding=UTF-8"/>
    <property name="driverClass" value="com.mysql.jdbc.Driver"/>
    </jdbc-connection-pool>
    <jdbc-resource enabled="true" jndi-name="bookstore" object-type="user" pool-name="mysql_bookstore_rootPool"/>
    </resources>
    </code></pre>

    Các bạn chú ý dòng này:

    <pre><code class="xml"><jdbc-resource enabled="true" jndi-name="bookstore" object-type="user" pool-name="mysql_bookstore_rootPool"/>
    </code></pre>

    Ta sẽ tạo một connection pool có tên là mysql_bookstore_rootPool (trùng với giá trị của thuộc tính pool-name) và một JDBC resource có tên là bookstore (trùng với giá trị của thuộc tính jndi-name) kết nối vào connection pool trên.

    Tạo connection pool

    Ở đây mình sẽ hướng dẫn các bạn tạo connection pool bằng console. Cách này có vẻ tỉ lệ thành công cao hơn cách dùng giao diện GUI của GlassFish.

    Các bạn vào trong thư mục C:\Program Files\glassfish-4.1\glassfish\lib, kiểm tra xem đã có thư viện mysql-connector-java chưa. Nếu chưa có, các bạn:

    • Vào tab Service -> Servers, tắt GlassFish đi.
    • Vào địa chỉ https://dev.mysql.com/downloads/connector/j/5.1.html, tải file mysql-connector-java-5.1.41.zip. Sau đó giải nén, copy file mysql-connector-java-5.1.41-bin.jar vào thư mục C:\Program Files\glassfish-4.1\glassfish\lib
    • Quay lại Service -> Servers, bật GlassFish.

    Vào thư mục C:\Program Files\glassfish-4.1.1\glassfish\bin, chạy file asadmin.bat. Nhập dòng lệnh sau:
    create-jdbc-connection-pool --datasourceclassname com.mysql.jdbc.jdbc2.optional.MysqlDataSource mysql_bookstore_rootPool để tạo connection pool có tên là mysql_bookstore_rootPool:

    Ta vào địa chỉ http://localhost:4848, chọn Resources -> JDBC -> JDBC Connection Pool kiểm tra xem connection pool đã được tạo chưa:

    OK, như vậy là tạo thành công!

    Click vào connection pool mysql_bookstore_rootPool. Trong tab General, ta chọn Resource Type là javax.sql.DataSource, rồi ấn nút Save để lưu:

    Nếu lưu thành công, các bạn chuyển sang tab Additional Properties, thêm các thuộc tính sau (các bạn ấn nút Add Property để thêm mới một thuộc tính):

    Trong đó:

    • User và Password là username và password đăng nhập MySQL.
    • URL là chuỗi kết nối ta đã tạo ở bước 2.

    Các bạn ấn nút Save để lưu lại.

    Quay lại tab General, ấn nút Ping để xem connection pool đã hoạt động được chưa. Nếu hiện lên thông báo Ping Succeedded thì là OK rồi:

    Tạo JDBC Resource

    Vẫn ở trong giao diện console, các bạn nhập dòng lệnh sau:
    create-jdbc-resource --connectionpoolid mysql_bookstore_rootPool bookstore để tạo JDBC Resource có tên là bookstore kết nối với connection pool mysql_bookstore_rootPool:

    Quay lại deploy module Demo-ejb, ta sẽ thấy deploy thành công.

    Nếu như vẫn không deploy được, các bạn thử mở file persistence.xml trong Configuration Files của Demo-ejb, chọn tab Source, kiểm tra xem giá trị trong thẻ <jta-data-source> có trùng với tên của JDBC Resource ta vừa tạo bên trên không (bookstore):
    <jta-data-source>bookstore</jta-data-source>

    Có một chú ý là, trong quá trình phát triển ứng dụng, ta chỉ cần deploy module EJB một lần duy nhất, vì Netbeans sẽ deploy hộ chúng ta khi ứng dụng có thay đổi.

    Bước 5: Tích hợp JSF vào Demo-war và chạy thử

    Tích hợp JSF

    Để thêm framework JSF vào module Demo-war, ta chuột phải Demo-war, chọn Properties. Trong tab Frameworks, ta add JavaServer Faces, rồi chọn OK:

    Chọn phiên bản JSF 2.2, rồi chọn OK:

    EJB và JSF

    Ở đây mình sẽ thử insert vào DB một tác giả có tên là Kim Dung và sẽ in ra màn hình tên của tác giả vừa insert đó.

    Bảng Author của mình chỉ gồm 2 trường là ID và name.

    Đầu tiên, ta tạo một Managed Bean có tên là Hello có nội dung như sau:

    <pre><code class="java">import entity.Author;
    import java.io.Serializable;
    import javax.annotation.PostConstruct;
    import javax.ejb.EJB;
    import javax.faces.bean.ManagedBean;
    import javax.faces.bean.SessionScoped;
    import sessionbean.AuthorFacadeLocal;
    @ManagedBean(name = "hello")
    @SessionScoped
    public class Hello implements Serializable {
    private Author author;
    @EJB
    AuthorFacadeLocal authorDAO;
    @PostConstruct
    public void init() {
    authorDAO.create(new Author(1, "Kim Dung"));
    author = authorDAO.find(1);
    }
    public Hello() {
    }
    public Author getAuthor() {
    return author;
    }
    public void setAuthor(Author author) {
    this.author = author;
    }
    }
    </code></pre>

    Trong file index.xhtml, mình sẽ in ra tên tác giả vừa insert:

    <pre><code class="html"><?xml version='1.0' encoding='UTF-8' ?>
    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
    <html xmlns="http://www.w3.org/1999/xhtml"
    xmlns:h="http://xmlns.jcp.org/jsf/html">
    <h:head>
    <title>Facelet Title</title>
    </h:head>
    <h:body>
    Hello from #{hello.author.name}
    </h:body>
    </html>
    </code></pre>
